>优先队列的  基本操作 ：`priority_queue<int> `

>```c++
>empty()  // 如果队列为空返回真
>
>pop()     //  删除队顶元素
>
>push()     //入队一个元素
>
>size()     // 返回优先队列中拥有的元素个数
>
>top()       // 返回优先队列队顶元素
>```
